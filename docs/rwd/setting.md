# 環境建立

```html
<!-- 瀏覽器顯示寬度 = 裝置寬度 -->
<!-- 縮放比為 1 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```

### 參數設定

| 參數          | 說明             |
| ------------- | ---------------- |
| device-width  | 畫度寬度         |
| device-height | 畫度高度         |
| initial-scale | 初始縮放比       |
| minimum-scale | 最小縮放比       |
| maximum-scale | 最大縮放比       |
| user-scalable | 使用者是否可縮放 |

```html
<meta
  name="viewport"
  content="width=device-width, initial-scale=1.0,user-scalable=no,maximum-scale=1.0,minimum-scale=1.0"
/>
```
